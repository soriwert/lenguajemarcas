Trabajo HTML

Inicio del documento <!Doctype>

Debe ser lo primero que debe ir en el documento, no es una etiqueta,
es una instrucción para el navegador que declara la versión de html
que se está usando. Se agrega html después para que el navegador sepa
que tipo de documento debe esperar.

Estructura
así --> <!Doctype html>


ESPECIFICACION DEL LENGUAJE <html lang="*">

Lo siguiente que debemos escribir (donde pone asterísco) es el lenguaje
que queremos que nuestra página web sea, en nuestro caso sería español.


CABECERA <head> </head>

Delimita la seccion de encabezamiento de un documento Html. Es imprescindible,
por que aquí se pone el nombre de la pagina, el idioma, las relaciones con hojas de estilos.
Debe englobar obligatoriamente la directiva <meta> y la directiva <tittle>.

Estructura
<tittle>Titulo del proyecto</tittle>
<meta charset = "Codificación "ANSI UTF-8...">
<meta name="author"content="Nombre del autor">


CUERPO<body> </body>

Delimita el cuerpo de un documento Html, la directiva<body> permite
agregar interactividad entre el sitio web y el visitante, cuando el visitante
o la pagina en si misma realiza una acción.

Estructura
<body></body>

HEADER
Estructura
<header></header>
Define la cabecera de una página o sección. Usualmente contiene un logotipo, el título del sitio Web y una tabla de navegación de contenidos.


SECTION
Estructura
<section></section>
crean un area, un contenedor, una seccion, un bloque se pueden introducir agrupaciones tematicas de los contenidos que incluye, mientras que los areas creadas


ASIDE
Estructura
<footer></footer>
Define algunos contenidos vagamente relacionados con el resto del contenido de la página. Si es removido, el contenido restante seguirá teniendo sentido


FOOTER
Estructura
<aside></aside>
representa el pie de una pagina Web donde se puede añadir contenido como: copyright, referencias, enlaces etc. Sin embargo, la directiva <footer> se puede utilizar como el pie de otras areas presentes en un documento Html como <section>, <article> etc...

CONTENIDO ENCAJADO: IMAGEN
Estructura
<img src= "www.....com/.img" width "ancho" height = "alto" alt = "nombre archivo">
<img> primero lo que hace es de buscar la imagen mediante la dirección definida (de nosotros) en el atributo @srcvinculo, luego si la encuentra, obtiene la información de la imagen (dimensiones, tipo de imagen) y reserva espacio necesario para imprimirla en la página Web.


COMANDO &+nbsp Estructura
&+nbsp (todo junto y sin simbolo +) es un comando especial en el cual sirve para juntar dos palabras a la hora de reducir una pagina web y pasarlas al siguiente parrafo a ambas a la vez.

Reduce la página hasta la palabra, escribas y perfecto. Sigue reduciendo y verás como ambas palabras pasan al nuevo parrafo por no dejarlas separarse. Escribas perfecto


TABLAS
Estructura
<table border="1">
<tr>
<th>Columna 1</th>
<th>Columna 2</th>
</tr>
<tr>
<td>celda a</td>
<td>celda b</td>
</tr>
<tr>
<td>celda c</td>
<td>celda d</td>
</tr>
</table>
<table>Es la directiva que delimita la tabla. El atributo @border que acompaña a la directiva, dibuja un borde alrededor de los elementos que componen la tabla. Cuanto mas mayor el valor numerico del atributo mas grueso el borde.
<tr>Se encarga de construir las filas de la tabla.
<th> y <td>Dentro de una fila puede haber una o mas columnas con datos; cada dato esta encerrado de la directiva
Tanto <th> como <td> contienen atributos como @rowspan y @colspan que permiten la expansion de las celdas a lo largo de filas y columnas
coldspan sirve para combinar filas. se coloca en td y se eliminan los demás para que solo reconozca uno.
rowspan sirve para combinar columnas
